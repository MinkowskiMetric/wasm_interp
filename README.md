## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/MinkowskiMetric/wasm_interp.git
```

2. Build

```sh
cargo build
```

3. Test

```sh
cargo test
```
